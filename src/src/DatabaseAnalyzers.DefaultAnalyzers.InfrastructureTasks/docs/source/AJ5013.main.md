Parameter references should use the same casing as the parameter declaration.

Non-compliant:

```tsql
CREATE PROCEDURE [dbo].[P1]
    @Param1 VARCHAR(MAX)
AS
BEGIN
    PRINT @PARAM1 -- Different casing compared to the parameter declaration @Param1
END
```

Compliant:

```tsql
CREATE PROCEDURE [dbo].[P1]
    @Param1 VARCHAR(MAX)
AS
BEGIN
    PRINT @Param1
END
```
