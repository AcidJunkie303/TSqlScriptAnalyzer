Tables should have a primary key.

Non-compliant:

```tsql
CREATE TABLE Table1
(
    Id          INT NOT NULL,
    Value1      NVARCHAR(128) NOT NULL
)
```

Compliant:

```tsql
CREATE TABLE Table1
(
    Id          INT NOT NULL PRIMARY KEY, -- not preferable because a random name will be generated
    Value1      NVARCHAR(128) NOT NULL
)
```

```tsql
CREATE TABLE Table1
(
    Id          INT NOT NULL,
    Value1      NVARCHAR(128) NOT NULL,
    CONSTRAINT  PK_Table1 PRIMARY KEY (Id)
)
```
