## Details

The first statement in a batch must be `USE` <database>.

**Non-compliant:**

```tsql
CREATE TABLE dbo.Table1 -- Should be 'USE <database-name>' instead
(
    Id            INT NOT NULL,
    Value1        NVARCHAR(128) NOT NULL
)
```

**Compliant:**

```tsql
USE MyDb
GO

CREATE TABLE dbo.Table1
(
    Id            INT NOT NULL,
    Value1        NVARCHAR(128) NOT NULL
)
```
