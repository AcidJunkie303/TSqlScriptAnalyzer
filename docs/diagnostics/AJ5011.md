## Details

Unreferenced parameters should be removed.

The following object types are checked:

- Procedures
- Functions

**Non-compliant:**

```tsql
CREATE PROCEDURE [dbo].[P1]
    @Param1 VARCHAR(MAX)
AS
BEGIN -- @Param1 is not referenced in the procedure body
    SELECT 1
END
```

**Compliant:**

```tsql
CREATE PROCEDURE [dbo].[P1]
    @Param1 VARCHAR(MAX)
AS
BEGIN
    PRINT @Param1
END
```
