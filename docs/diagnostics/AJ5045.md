## Details

Enforcing new-lines before/after `GO` statements.

**Non-compliant (if newline is required after `GO`):**

```tsql
GO
PRINT 'Hello' --should be an empty line
```

**Compliant (if newline is required after `GO`):**

```tsql
PRINT 'Hello'
GO

PRINT 'World'
```

```tsql
PRINT 'Hello'
GO
-- some comment
PRINT 'World'
```
