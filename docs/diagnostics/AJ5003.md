## Details

Assuming, the script originate from a directory for database 'MyDd', no other database name than 'MyDb' should be used.

**Non-compliant:**

```tsql
USE OtherDatabase
USE [OtherDatabase]
```

**Compliant:**

```tsql
USE MyDb
USE [MyDb]
```
