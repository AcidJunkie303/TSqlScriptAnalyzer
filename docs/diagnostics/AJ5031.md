## Details

Redundant pair of parentheses.

**Non-compliant:**

```tsql
IF ((1=1))
BEGIN
    PRINT 'Hello'
END
```

**Compliant:**

```tsql
IF (1=1)
BEGIN
    PRINT 'Hello'
END
```
