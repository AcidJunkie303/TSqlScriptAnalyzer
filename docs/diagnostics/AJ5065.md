## Details

~~~~Alias reference differs in casing

**Non-compliant:**

```tsql
SELECT      SOURCE_TABLE.Id  -- alias 'SOURCE_TABLE' differs from its declaration below 'source_table'
FROM        Table1 source_table
```

**Compliant:**

```tsql
SELECT      source_table.Id
FROM        Table1 source_table
```
