## Details

Do not separate `SET` option statements by `GO`.

Non-compliant:

```tsql
SET ANSI_NULLS ON

GO -- unecessary GO statement

SET ARITHABORT ON
```

Compliant:

```tsql
SET ANSI_NULLS ON
SET ARITHABORT ON
GO
```
