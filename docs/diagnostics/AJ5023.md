## Details

Statement should begin on a new line.

****Non-compliant:**

```tsql
SET @x = 0; IF (@y < 0)  -- IF statement should be on a new line
BEGIN
    SET @y = 0
END
```

**Compliant:**

```tsql
SET @x = 0;
IF (@y < 0)
BEGIN
    SET @y = 0
END
```
