## Details

Multiple consecutive `GO` statements.

**Non-compliant:**

```tsql
PRINT 'Hello'

GO

GO

PRINT 'World'
```

```tsql
PRINT 'Hello'

GO
/* comment */
-- comment
GO

PRINT 'World'
```

**Compliant:**

```tsql
PRINT 'Hello'

GO

PRINT 'World'
```
