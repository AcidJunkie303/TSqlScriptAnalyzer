## Details

Do not use specific, banned functions.

Non-compliant (when `GETDATE()` is banned):

```tsql
DECLARE @now DATETIME = GETDATE() -- GETDATE() is banned
```

Compliant (when `GETDATE()` is banned):

```tsql
DECLARE @now DATETIME = GETUTCDATE()
```
