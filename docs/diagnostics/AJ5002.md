# Details

Non-Compliant:

```tsql
SET @x = N'a' + 'b' -- combining unicode and ASCII strings
SET @y = 'a' + N'b' -- combining ASCII and unicode strings
```

Compliant:

```tsql
SET @x = 'a' + 'b' -- combining ASCII strings
SET @y = N'a' + N'b' -- combining unicode strings
```
