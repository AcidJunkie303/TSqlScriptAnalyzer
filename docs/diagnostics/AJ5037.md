## Details

Object creation without explicit schema name.

Non-compliant:

```tsql
CREATE VIEW V1 -- no schema name specified
AS
SELECT 1 AS Column1
```

```tsql
CREATE PROCEDURE P1 -- no schema name specified
AS
BEGIN
    SELECT 1 AS Column1
END
```

Compliant:

```tsql
CREATE VIEW dbo.V1
AS
SELECT 1 AS Column1
```

```tsql
CREATE PROCEDURE dbo.P1
AS
BEGIN
    SELECT 1 AS Column1
END
```
