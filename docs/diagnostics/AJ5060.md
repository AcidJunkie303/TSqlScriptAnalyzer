  <tr>
    <td class="header"><b>Insertion string {1}</b></td>
    <td>Reserved Word</td>
  </tr>

## Details

Do not use reserved words or keywords for object names. The following object types are checked:

- Column Names
- Function Names
- Procedure Names
- Table Names
- View Names
