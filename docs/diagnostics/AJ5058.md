## Details

Restricts drop statements to certain file name patterns.


## Settings

```json
{
  "Diagnostics": {
    "AJ5058": // Drop statements
    {
      "AllowedInFilesByDropStatementType": {
        "DropTable": [
          "*__drop__*.sql"
        ],
        "DropProcedure": [
          "*__drop__*.sql"
        ],
        "AlterTableDropTableElement": [
          "*__drop__*.sql"
        ]
      }
    }
  }
}
```
