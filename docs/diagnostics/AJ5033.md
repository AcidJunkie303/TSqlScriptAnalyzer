## Details

Do not nest ternary operators.

**Non-compliant:**

```tsql
SELECT IIF(@a=1, 'Hello', IIF(@b=1, 'world','there')) -- nested 'IFF' statement
```

**Compliant:**

```tsql
SELECT IIF(@a=1, 'Hello', 'world')
```
