## Details

Delete dead code.

Non-compliant:

```tsql
PRINT 'Hello'
THROW 60000, 'ooops', 1;
PRINT 'World' -- will never be executed
```

```tsql
WHILE(@a > 1)
BEGIN
    PRINT 'Hello'
    CONTINUE
    PRINT 'World' -- will never be executed
END
```

```tsql
GOTO MyLabel
PRINT 303 -- will never be executed

MyLabel:
```
