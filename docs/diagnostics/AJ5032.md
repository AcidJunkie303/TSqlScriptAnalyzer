## Details

Usage of non-standard comparison operator `!=`.

**Non-compliant:**

```tsql
IF (1 != 2) -- '!=' is not standard. Use '<>' instead
BEGIN
    PRINT 'Hello'
END
```

**Compliant:**

```tsql
IF (1 <> 2)
BEGIN
    PRINT 'Hello'
END
```
