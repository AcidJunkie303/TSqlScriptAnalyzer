## Details

Variable references should use the same casing as the declaration.

**Non-compliant:**

```tsql
DECLARE @Var1 INT = 303
SELECT @VAR1 -- @var has different casing compared to the declaration @Var1
```

**Compliant:**

```tsql
DECLARE @Var1 INT = 303
SELECT @Var1
```
